# League-Loading-Pauser

<img width="720" height="380" alt="Screenshot of the program" src="https://github.com/user-attachments/assets/c5102bf4-ac3e-4020-ac5c-85859d94f149" />
<br>
League Loading Pauser allows you to stay in the loading screen for longer by blocking the connection to league temporarily. Kinda useful when you need a quick bathroom break or whatever.
<br><br>

It works by creating a Windows Firewall rule using Powershell commands, which block the outbound connection to league's game exe file.
**(It doesn't affect the client)**
<br><br>
Pretty simple!
